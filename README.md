# Usage
`make` to compile.

```
./fractol julia
./fractol mandelbrot
./fractol sierpinski
```

Use `--fullscreen` to set the window in fullscreen mode, default window is 500x500

Controls:

`WASD` or `▲ ◄ ▼ ►` to move

`⏎` to change between fractals

`+` or `-` to increase or decrease the iterations

`1 2 3 4` to change Julia set

`Mouse wheel` to zoom in/out
