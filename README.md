# Usage
`make` to compile.

`mak exe` to execute the program.

or 

```
./fractol julia
./fractol mandelbrot
```

Controls:

`WASD` or `▲ ◄ ▼ ►` to move

`⏎` to change between fractals

`+` or `-` to increase or decrease the iterations

`Mouse wheel` to zoom in/out
